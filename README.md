# Aurora

A clean, glassy SaaS UI kit — blue, Inter, light + dark. Part of the
[UIKit](https://uikit.dev) gallery.

Tokens · React components · blocks · dashboard + landing templates · a bundled
Claude Code skill so AI builds **with** the kit.

➡️ **[USAGE.md](./USAGE.md)** to get started. Manifest: [`uikit.json`](./uikit.json).

```bash
git clone https://github.com/uikit-studio/aurora-uikit my-app
cd my-app && npx uikit-studio init
# then ask Claude Code: "build a billing dashboard using this kit"
```
